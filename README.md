# 🌱 DWC Grower Edition

**Der kostenlose Hydroponik-Nährstoffrechner für DWC-Grower.**

## Features
- 🧪 **9 Dünger-Marken** — Hesi, Canna Aqua, Plagron Hydro, GHE Flora Series, Advanced Nutrients, Biobizz, Aptus, House & Garden, Atami B'cuzz
- 📊 Automatische Dosierung in ml — nach Phase & Woche
- ⚠️ pH/EC Warnsystem mit Empfehlungen
- ⏳ Ernte-Countdown + Wechselwasser-Erinnerung
- 📝 Grow Diary lokal gespeichert
- 🤖 Gemini KI-Assistent (optional, eigener API-Key)
- 📱 Tablet & Mobile optimiert

## Nutzung
1. [**index.html herunterladen**](index.html)
2. Im Browser öffnen
3. Fertig — kein Server, kein Login, keine Cloud

Alles läuft 100% lokal. Keine Daten werden gesendet.
